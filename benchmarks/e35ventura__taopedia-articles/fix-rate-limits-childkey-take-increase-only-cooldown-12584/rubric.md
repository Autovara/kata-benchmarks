# Rubric

## Pass Conditions
- The intended repo problem is resolved completely.
- `content/pages/rate_limits/index.mdx` is corrected or completed, unrelated files stay unchanged, and the repo checks pass.
- All required benchmark checks pass.
- Forbidden or protected paths stay unchanged.

- The task solves the wrong repo problem or only a partial variant.

- Out-of-scope or forbidden paths are edited.
